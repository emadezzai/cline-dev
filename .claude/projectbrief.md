# Cline - Project Brief

## Overview
Cline هو امتداد VS Code يعمل كمساعد برمجة مستقل يعمل بالذكاء الاصطناعي. يستطيع إنشاء وتعديل الملفات، تشغيل الأوامر في الطرفية، استخدام المتصفح، والمزيد.

## Core Purpose
توفير تجربة تطوير برمجي تعتمد على الذكاء الاصطناعي مع الحفاظ على السيطرة الكاملة للمستخدم في كل خطوة.

## Key Features
- إنشاء وتعديل الملفات مع عرض الفروقات (diff view)
- تنفيذ الأوامر في الطرفية
- استخدام المتصفح للتفاعل مع المواقع
- دعم Model Context Protocol (MCP) لإضافة أدوات مخصصة
- دعم مزودين متعددين للذكاء الاصطناعي (Anthropic, OpenAI, Gemini, OpenRouter, AWS Bedrock, إلخ)
- تتبع التكلفة وعدد الـ tokens
- Checkpoints للمقارنة والاستعادة

## Target Users
- المطورون الذين يريدون مساعدة ذكية في البرمجة
- المطورون المبتدئون الذين يحتاجون تفسيرات وتعليم
- فرق التطوير التي تستخدم البرمجة المستقلة

## Success Metrics
- سهولة الاستخدام والبساطة
- دقة التنفيذ
- سرعة الاستجابة
- دعم نماذج متعددة
- أمان التحكم البشري
